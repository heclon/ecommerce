# Welcome to @ecommerce/ui

# Get started

Ecommerce Design System is a reusable components library that helps ecommerce contributors build UIs faster.
The goal is to make building durable UIs more productive and satisfying.

## **Install**

ecommerce components are written in React, and its stories are written in [Component Story Format](https://medium.com/storybookjs/component-story-format-66f4c32366df).

Add @ecommerce/ui to your project.

`yarn add @ecommerce/ui`

### **Use**

Import components you want into your UI

`import { Button } from '@ecommerce/ui';`

and use them like so

```ts
const example = () => (
  <div>
    <Button>Submit</Button>
  </div>
);
```
